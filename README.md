# Tip-Calculator
I made a tip calculator built with HTML, CSS, and Javascript. It lets you enter a bill amount and tip percentage for it to calculate the tip and total. I made it because I wanted to learn javaScript by building a project and applying the concepts I learned. I made it by using functions, DOM manipulation, and conditions to make the page interactive. The hardest part wasn't any one concept but trying to keep track of it all. There were so many things I was learning that I was forgetting a lot. I learned and applied the practice of debugging. Most of the time the reason my code wasn't working was because of a typo!
